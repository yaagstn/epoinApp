# Aplikasi Pencatat Poin Pelanggaran (E-Poin App)

![GitHub top language](https://img.shields.io/github/languages/top/yaagstn/epoinApp)
![GitHub last commit](https://img.shields.io/github/last-commit/yaagstn/epoinApp)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Aplikasi ini dirancang untuk mempermudah manajemen dan pencatatan poin pelanggaran siswa secara digital. Proyek ini bertujuan menggantikan sistem pencatatan manual yang rentan terhadap kesalahan data dan inefisiensi.

---

##  Fitur Utama

* **Pencatatan Cepat:** Input pelanggaran dan poin terkait secara *real-time*.
* **Database Terpusat:** Penyimpanan data pelanggaran dan poin secara aman.
* **Histori Pelanggaran:** Melihat riwayat lengkap pelanggaran yang dilakukan oleh siswa tertentu.
* **Sistem Peringatan Otomatis:** Pemberitahuan kepada wali murid atau pihak sekolah ketika poin pelanggaran mencapai batas tertentu.
* **Otentikasi Pengguna:** Sistem *login* untuk Guru/Wali Kelas dan Administrator.

---

##  Teknologi yang Digunakan

Aplikasi ini dikembangkan menggunakan tumpuan teknologi berikut:

* **Platform:** Android / Mobile
* **Bahasa Pemrograman:** Java (atau Kotlin, jika relevan)
* **Database:** Firebase / SQLite (Pilih salah satu yang Anda gunakan)
